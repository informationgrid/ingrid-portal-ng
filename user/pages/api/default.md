                                    STA MQTT-Broker
                                </h2>
                                <p>
                                    MQTT ist ein Internetprotokoll für die zeitnahe Bereiststellung von Echtzeit- und Sensordaten. MQTT ist Teil der Implementierung der SensorThings API.
